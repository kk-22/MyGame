# MyGame

## 開発理由
* Kotlin、Realmの学習
* 何度でも遊べるゲームの作成
* 趣味

## 仕様技術
* スクレイピング(数独のインポートに使用)
* Realm(データベース)
* GoogleAppsScript(簡易バックエンドとして使用)

## 機能
* 数独ゲーム
* シミュレーションゲーム(開発途中)
